 🃏 Card

Projeto **Card** desenvolvido com **HTML e CSS**, com o objetivo de criar um componente de cartão moderno e reutilizável. O foco do projeto é trabalhar layout, alinhamento e estilização visual utilizando apenas HTML e CSS.

📌 Sobre o Projeto

O **Card** é um componente visual que apresenta informações dentro de um cartão organizado, podendo conter imagem, título, descrição e botão. O design foi criado para ser simples, limpo e reutilizável em diferentes projetos.

Este projeto foi desenvolvido para praticar:

* Estruturação com HTML
* Estilização com CSS
* Criação de componentes UI
* Layout organizado
* Design moderno

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3

## 🎯 Funcionalidades

* Cartão estilizado
* Layout centralizado
* Espaçamento organizado
* Botão de ação
* Design moderno
* Estrutura reutilizável
